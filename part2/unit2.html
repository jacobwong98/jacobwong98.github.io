JavaScript

With HTML, we can determine the structure and content for web pages.

With CSS, we can determine the look of the content for web pages.

With Javascript, we can enhance the functionality/appearance of web pages. Despite the name, JavaScript is not the same as Java since one is a scripting language while the latter is object-oriented.

Much like CSS, we have three methods of applying JavaScript to our webpages:

  • : <input type="button" onclick="alert('Hello!')" value="Say Hello!">
  • : The code for our JavaScript goes in the "script" tag and is placed in the head or body section of an HTML document
  • : Instead of placing the script tag directly within the HTML document we can save the code in a .js file and add <script src="myScript.js" type="text/javascript"></script> to reference the script we want to use

Now that we can add JavaScript to our web pages, let's look at how make scripts.

Common Features/Rules for JavaScript

  • : This object represents the HTML document currently displayed in the browser. The document object itself has many helpful methods & attributes like
    • writeln - Adds text to HTML document (May not appear if not proper format)
    • getElementById - Like the name says, it returns the element based on the ID provided
    • getElementsByClassName - Returns element(s) based on the name of the class provided
    • getElementsByTagName - Returns element(s) based on the tag name (i.e. p, span, h1, input, etc).
  • Each statement SHOULD end with a semicolon (statement terminator) but JavaScript does not require them. Also JavaScript is case sensitive.
  • To declare a variable, add the keyword "var" before the name of your variable. (NOTE: We don't need to specify the type for the variable because the JavaScript interpreter will handle that)
    • Can use single or double quotes as long as there are matching quotes.
    • Can concatenate (combine) strings using the + operator. E.g. "H" + "i" becomes "Hi"
    • Can also combine mathematical operations with string concatenation.
      Caution: "y + 2 = " + y + 2 if y=5 will become "y + 2 = 52" vs. "y + 2 = " + (y + 2) if y=5 will become "y + 2 = 7" because the calculations/concatenation follow order of operations (BEDMAS).
  • JavaScript can perform Addition (+), Subtraction (-), Multiplication (*), Division (/) and Modulo (%, returns remainder)
  • JavaScript supports selection statements (if, if...else, switch) and repetition statements (while, do...while, for, for...in)

Functions

With functions like many other programming languages, you can reuse code that will be used multiple times to avoid repeating code. The syntax for creating a function is:

function myFunction(input){ Code goes here }

Much like variables, we don't need to specify the type for the input or output parameters since the JavaScript interpreter will handle them.
Great, now we can reuse code but can we perform functions when a button is clicked, a dropdown changes value or maybe validate user input?
The answer is of course and this can be done using event handling

To add event handling, there are 2 items we need:
  1. Any HTML element such as document, p, input, button or window (element that represents the browser window).
    To get any element that isn't document or window (both always available), we can use document object's getElement methods to get element(s) we want.
  2. Once we have the element we can then add event handling to that element: element.addEventListener(event, function, true value);
    • would be the type of event that you want to trigger the listener (e.g. change, click, load, blur, keyup, etc)
    • is the function that runs when the event is detected
    • typically is false (value deals with how events are handled based on DOM tree)

JavaScript also provides some objects that can assist in improving the functionality of your webpages. Each object has its own attributes and methods (difference between methods and functions is that methods require the name of its object followed by the invocation of the method).

  • performs common mathematical calculations (e.g. Math.pow(2,3) = 23)
  • deals with series of characters treated as a single unit
  • handles date and time manipulations (e.g. getDate method will return the day of the month)

Introduction to canvas

The HTML canvas element allows you to draw 2D graphics/animations, font/image manipulation and inserting images/videos onto this element. The benefit of using canvas is that it's built into the browser so no plug-ins are needed.
To draw on the canvas, we need to understand the coordinate system it follows

  • Origin (0, 0) is at the top-left corner of the canvas
  • X-axis increases positively to the right of the screen
  • Y-axis increases positively towards the bottom of the screen
